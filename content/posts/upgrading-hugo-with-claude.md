I think I found my new favourite thing about Claude Code: using it to upgrade
Hugo. Switching to a static blog generator was supposed to make things easier,
but given how infrequently I actually write anything here, every time I sat
down I'd fight with the latest version of Hugo for a while and sometimes just
give up. (Of course I probably didn't _really_ need to upgrade Hugo,
especially since I Dockerized it... but I have some sort of weird obsession
to keep my systems and tools always up to date).

Anyway Claude managed to upgrade it, switch Docker images, and sort out
theme errors (that's probably the biggest annoyance since the themes aren't
always up to date with the latest Hugo changes). The best part is that when
I got even lazier and asked it to create a new post, intending for it to just
create the new file and header, it took my topic and wrote a big
self-aggrandizing article about how great Claude is at upgrading Hugo. But this
isn't OpenClaw so all you get is my rambling.
